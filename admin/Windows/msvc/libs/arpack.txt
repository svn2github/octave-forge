Package: ARPACK
Version: 96
Home page: http://www.caam.rice.edu/software/ARPACK/
Download: http://www.caam.rice.edu/software/ARPACK/SRC/arpack96.tar.gz
          http://www.caam.rice.edu/software/ARPACK/SRC/patch.tar.gz
License: BSD-like

Compilation: from MinGW shell with VS environment + f2c + cc-msvc

Requirement: cc-msvc, f2c, libf2c

1) Uncompress arpack96.tar.gz

2) Uncompress patch.tar.gz in the same directory (this will overwrite
   several files)

3) Copy arpack.makefile to ARPACK source tree

4) make -f arpack.makefile

5) Install (copy manually):
	. arpack.dll into bin directory
	. arpack.lib into lib directory
