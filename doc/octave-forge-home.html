The GNU Octave Repository is a central location for the collaborative development of packages for GNU Octave.

The octave-forge package contains the source for all the functions plus build and install scripts. These are designed to work with the latest development version of Octave (available from http://www.octave.org/download.html), but most functions will work on earlier versions of Octave including 2.0.x versions.

For a list of octave and octave-forge functions see the categorical index. (For a local copy, download and expand http://octave.sf.net/index.tar.gz).

If you have code to contribute, send a message to the repository mailing list or join the development team.
