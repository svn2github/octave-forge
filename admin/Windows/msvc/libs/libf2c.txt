Package: libf2c
Version: none
Home page: http://www.netlib.org/f2c/
Download: http://www.netlib.org/f2c/libf2c.zip
License: Free

Compilation: from Windows command shell with VS environment

Requirement: none

1) Edit makefile.vc and add "-MD -DIEEE_COMPLEX_DIVIDE" to the CFLAGS variable

2) Edit fio.h and comment the line "extern int isatty(int);" (using /* */ pair)

2) nmake -f makefile.vc

3) Copy vcf2c.lib as f2c.lib into your lib directory
